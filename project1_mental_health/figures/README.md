## Exploratory Data Analysis (EDA)

### `class_distribution.png`
**What it shows:** Histogram of target variable (treatment-seeking behavior)
- X-axis: Did not seek treatment | Sought treatment
- Y-axis: Count
- **Key insight:** Class imbalance (63% vs 37%) visible; SMOTE will be used to balance

### `feature_distributions.png`
**What it shows:** Distribution plots for all continuous and categorical features
- Shows patterns in age, company size, years at current job, etc.
- Identifies potential outliers or data quality issues

### `correlation_heatmap.png`
**What it shows:** Pearson correlation between all features and target variable
- Color scale: Dark blue (negative) to dark red (positive)
- Sorted by correlation strength
- **Key features:** work_interfere (0.42), family_history (0.28), benefits (0.25)

### `missing_values_heatmap.png`
**What it shows:** Percentage of missing data per feature
- Identifies which fields have data quality issues
- Informs imputation strategy

## Model Evaluation

### `model_comparison.png`
**What it shows:** ROC curves for all 4 models on validation set
- X-axis: False Positive Rate
- Y-axis: True Positive Rate
- Lines: Logistic Regression, Random Forest, XGBoost, Neural Network
- **Key metric:** Logistic Regression achieves AUC=0.723 (best discriminator)

### `confusion_matrix_logistic_regression.png`
**What it shows:** 2×2 confusion matrix for selected model
- True Negatives (TN): Correctly predicted "no treatment"
- False Positives (FP): Incorrectly predicted "treatment"
- False Negatives (FN): Incorrectly predicted "no treatment"
- True Positives (TP): Correctly predicted "treatment"
- **Derived metrics:** Sensitivity, Specificity, Precision, Recall

### `model_metrics_table.png`
**What it shows:** Side-by-side comparison of all 4 models
- Columns: AUC-ROC, Accuracy, Precision, Recall, F1-Score
- Rows: Logistic Regression, Random Forest, XGBoost, Neural Network

## Explainability & Interpretability

### `shap_feature_importance.png`
**What it shows:** SHAP (SHapley Additive exPlanations) summary plot
- Each dot = one prediction
- Horizontal position = contribution to prediction
- Color = feature value (red = high, blue = low)
- **Insight:** work_interfere pushes predictions in positive direction strongest

### `shap_waterfall_example.png`
**What it shows:** Detailed SHAP breakdown for a single prediction
- Base value: Average model output across all data
- Feature contributions: How each feature shifts the prediction
- **Example:** Shows why a specific person was predicted to seek treatment

### `feature_importance_traditional.png`
**What it shows:** Feature coefficients from Logistic Regression
- X-axis: Coefficient magnitude
- Positive coefficients → increase probability of treatment-seeking
- Negative coefficients → decrease probability
- **Top factors:** work_interfere (+0.87), family_history (+0.52), benefits (+0.41)

## Fairness & Bias Analysis

### `fairness_audit_by_gender.png`
**What it shows:** AUC-ROC performance broken down by gender
- X-axis: Gender (Male, Female, Non-binary)
- Y-axis: AUC-ROC score
- Error bars: 95% confidence intervals
- **Result:** No statistically significant difference (all CIs overlap)

### `fairness_audit_by_age.png`
**What it shows:** AUC-ROC performance broken down by age groups
- 18-29, 30-44, 45-60, 60+
- **Finding:** Age groups show similar performance; no significant disparity

### `demographic_parity_matrix.png`
**What it shows:** Statistical tests for fairness across groups
- Rows: Different protected attributes (age, gender)
- Columns: Fairness metrics (demographic parity, equal opportunity)
- Values: P-values from statistical tests
- **Threshold:** p > 0.05 means "fair" (no significant disparity)
