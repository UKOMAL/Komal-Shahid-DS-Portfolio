Applied survey ML (OSMI 2016)

Binary outcome prediction on the OSMI 2016 tech-industry survey (N=1,259). Four-classifier comparison, SHAP explainability, bootstrapped fairness audit by gender and age, and a novel Employer Support Index (ESI). Best model: Logistic Regression at AUC 0.723. Write-up, figures, and code are in the portfolio repo as DSC680 Project 1.

Key contributions
  • Custom Employer Support Index — novel composite feature engineering
  • SHAP waterfall plots for per-prediction explainability
  • SMOTE oversampling for class imbalance correction
  • Bootstrapped fairness audit across gender and age subgroups
  • 4-model comparison: Logistic Regression, Random Forest, SVM, XGBoost
Why it matters

Workplace wellness and benefits uptake are hard to measure at scale. This model surfaces which employer-side factors align with self-reported care-seeking on the survey, giving HR and policy teams an actionable, explainable view — not a black box.

• Anomaly Detection · Finance AI

Ensemble Fraud Detection System

Financial fraud detection on a highly imbalanced real-world transaction dataset (fraud rate: 0.27%). A three-model ensemble — PyTorch Autoencoder (reconstruction-error scoring), Isolation Forest, and XGBoost — with SHAP feature attribution for explainability. Primary metric: AUPRC to handle extreme class imbalance.

Ensemble models
Key contributions
  • Autoencoder trained on normal transactions; anomalies flagged by high reconstruction error
  • Isolation Forest for unsupervised outlier detection as a complementary signal
  • XGBoost trained on labeled fraud/non-fraud with SMOTE for class rebalancing
  • Ensemble score fusion via weighted probability averaging
  • SHAP waterfall plots for per-prediction explanation — critical for compliance
  • AUPRC used as primary metric (F1 and ROC alone mislead on 0.27% imbalance)
Why it matters

Financial fraud detection requires two things simultaneously: high recall (catch fraud) and high precision (avoid false accusations). Ensemble methods outperform any single model here because the autoencoder catches novel unseen patterns while XGBoost captures known fraud signatures — and SHAP makes the decisions auditable for regulators.
