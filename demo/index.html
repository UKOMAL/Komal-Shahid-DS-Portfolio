About This Demo

This interactive demonstration shows how federated learning enables collaborative AI model training while preserving privacy. In a real healthcare setting, patient data would remain securely within each institution, but for this demo, we simulate the process using a pre-trained model running entirely in your browser.

The example below uses a simplified model to detect potential heart conditions from ECG data. You can try it with example data or upload your own synthetic ECG samples.
