What is anamorphic 3D?

Anamorphic illusions flatten 3D intent into a 2D image so that, from one vantage point, content seems to leap out of the screen — like Seoul-style LED waves. Here the math is explicit: estimate per-pixel depth, then push pixels outward along rays from the frame center, proportional to depth × viewing geometry.

Why it matters

Manual anamorphic frames for large LED façades are slow and expensive. Automating the geometric stage (depth + warp + graded color) makes iteration cheap enough for creative and ad-tech workflows — this lab shows the core effect; Python adds batching and optional neural depth.

Fast feedback

Honest stack
