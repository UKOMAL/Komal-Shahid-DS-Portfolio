01 — About

I work at the intersection of data, ethics, and care.

I'm a data scientist focused on applied machine learning for healthcare and human-centered problems — places where getting the model right is only half the work, and the other half is making sure what you built is fair, explainable, and actually used.

Technical rigor is the floor, not the ceiling. A model that isn't interpretable, private, and fair isn't finished.

DSC680 Project 1 is a four-classifier pipeline on the OSMI 2016 tech-industry survey (N=1,259): a novel Employer Support Index, SHAP explainability, and a bootstrapped fairness audit across gender and age. Best AUC: 0.723.

• DSC680 · Project 1

Applied survey ML (OSMI 2016)

Binary outcome prediction on the OSMI 2016 tech-industry survey (N=1,259). Four-classifier comparison, SHAP explainability, bootstrapped fairness audit by gender and age, and a novel Employer Support Index (ESI). Best model: Logistic Regression at AUC 0.723. Write-up, figures, and code are in the portfolio repo as DSC680 Project 1.

Key contributions
  • Custom Employer Support Index — novel composite feature engineering
  • SHAP waterfall plots for per-prediction explainability
  • SMOTE oversampling for class imbalance correction
  • Bootstrapped fairness audit across gender and age subgroups
  • 4-model comparison: Logistic Regression, Random Forest, SVM, XGBoost
Why it matters

Workplace wellness and benefits uptake are hard to measure at scale. This model surfaces which employer-side factors align with self-reported care-seeking on the survey, giving HR and policy teams an actionable, explainable view — not a black box.

Key features

  • Three distinct 3D effects: Shadow Box, Screen Pop, Anamorphic
  • Random Forest ensemble for commercial performance prediction
  • Real-time image processing pipeline
  • Statistical optimization across 1,000+ simulated scenarios
Findings

Contrast is the primary predictor of commercial success (importance 0.35). Optimal viewing angles vary by illusion type — with measurable lifts in brand recall (156%) and a 67% reduction in return rates for fashion retail.

• Applied NLP

Notely — AI-Powered Note Taking

Intelligent note management built with Streamlit and LLM APIs. AI-powered summarization, semantic search across notes, smart template generation, and PDF ingestion — designed for real-world daily productivity.

Key features
  • AI summarization using OpenAI API for long-form notes and PDFs
  • Semantic search across stored notes using embedding similarity
  • Smart template generation for recurring note types
  • PDF processing and text extraction pipeline
  • Clean Streamlit UI optimized for low-friction daily use
Design philosophy

Notely is built around the principle that AI assistance should feel invisible — enhancing your workflow without demanding new habits or complex UX. Notes should be captured fast; intelligence surfaces when you need it.
